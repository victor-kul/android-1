Ссылка на github - репозиторий с файлами для первого урока по курсу Android-1, с некоторым экспериментальным интерфейсом:

https://github.com/victor-kul/android-1

Описание:

Приложение представляет собой форму с возможностью случайного переключения выдаваемых на экран, заранее заложенных фраз, по нажатию кнопки. Также в форме есть ещё пару кнопок, на которые временно навешено всплывающее уведомление, оповещающее момент соответсвующего нажатия.
